ローマ字 上計画について

「ローマ字 上」では、50音をローマ字で勉強し、 その過程で、すべてのアルファベットも覚えていきます。 開発の現場で使われているほぼすべてのプログラミング言語は、 アルファベットを使ってできていますし、 英語等の学習にもつながっていきますので、これを機会に覚えていきましょう。 小学生高学年くらいのお子様が始めるなら、 小学校でもローマ字を勉強していると思います。すでに十分理解している状態でしたら、 ローマ字上下は飛ばしてもらっても大丈夫です。

勉強内容

まずは、ヘボン式ローマ字の大文字を、順番に勉強していきます。 一通り勉強した後、アルファベットの順番と、小文字を勉強します。

教え方

声にだしておぼえるようにしましょう。

ホワイトボードや、せんせいをお持ちなら、遊びながら勉強することができます。

チェック項目

  • すべてのアルファベット大文字の読み書きができる
  • すべてのアルファベット小文字の読み書きができる
  • すべてのローマ字読み書きができる
