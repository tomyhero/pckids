# Slackに登録しよう

## 概要

授業内でのやりとり、家からの質疑応答に利用するチャットツール Slackに登録、インストールします。

## Slackとは

無料で活用できるチャットツールです。
ブラウザーだけでも利用できますが、 アプリケーションをインストールしての利用をおすすめしています。


## 登録

登録には招待を受ける必要があります。
招待メールを送りますので、手続きにそって登録してみてください。


## pckidsへの参加

pckidsというチームに参加しましょう。
ブラウザーでは、 http://pckids.slack.com/ でアクセスできます。
